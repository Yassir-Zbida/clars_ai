# clars.ai

CRM Intelligent pour Freelancers & Solopreneurs. Stack: Next.js 14 · MongoDB (Mongoose) · Gemini · Whop.

## Setup

1. Copy `.env.example` to `.env.local` and fill in values (at least `MONGODB_URI` and `NEXTAUTH_SECRET` for full features).
2. `npm install`
3. `npm run dev`

## Scripts

- `npm run dev` — development server
- `npm run build` — production build

See `cursorrules` for code style, architecture, and feature rules.
